# Mini World

## Description

A mini game website which includes Tic Tac Toe, Rock Paper Scissors, Maze, and Snake (only available in PC). The games are built in JavaScript.

## Build status

Started on: 21 Apr 2020 <br>
Completed on: 4 May 2020 <br>
